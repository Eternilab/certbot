# Description

Ce rôle installe et configure certbot.
Certbot est un utilitaire permettant de générer des certificats via ACME.

# Paramètres

acme_server: Url du serveur ACME

# Fichiers modifiés par le rôle 

/etc/letsencrypt/cli.ini

# Compatibilité

Ce rôle est compatible Debian
